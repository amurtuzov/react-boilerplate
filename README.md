# react-ts boilerplate

## Версия Node - v18.12.1

## Build
```
yarn
yarn build
```

## Build Preview
```
yarn build
yarn preview
```

## Develop mode
```
yarn serve
```

### Kill 3000 and 8080 ports processes
```
yarn kill
```

### Lint Fixes
```
yarn lint
```

### Type checks
```
yarn typecheck
```
